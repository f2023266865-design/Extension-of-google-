Privacy Policy

Last updated: March 12, 2026

AI Learning Copilot

1. Overview

AI Learning Copilot (“the Extension”) is a Chrome browser extension that provides AI-powered explanations of web page content. Your privacy is important to us. This policy describes what data we collect, how we use it, and how we protect it.

2. Data We Collect

  • Page content (transient): When you click “Explain This Page,” the visible text of the current tab is extracted and sent to our backend for AI processing. This content is not stored permanently.
  • Email address: If you choose to sign in, your email is used for authentication via Supabase and to manage your subscription.
  • Usage counts: We track daily explanation counts per user to enforce free-tier limits and provide usage analytics.
  • Explanation history: Explanations are stored locally in your browser (chrome.storage.local) and are never uploaded to external servers.
  • API keys: If you connect a third-party AI provider, your API key is encrypted with AES-256-GCM and stored locally. It is never transmitted to our servers.

3. Data We Do NOT Collect

  • We do not collect browsing history or track which websites you visit.
  • We do not sell, share, or monetise any personal data.
  • We do not use cookies or third-party analytics trackers in the extension.
  • We do not store the raw page content after generating an explanation.

4. Third-Party Services

  • Supabase: Used for user authentication and subscription record-keeping. Subject to the Supabase Privacy Policy.
  • Stripe: Used for payment processing. Subject to the Stripe Privacy Policy.
  • OpenAI / Google / Anthropic / OpenRouter: If you connect an AI provider, page text is sent to that provider’s API for processing, subject to the provider’s own privacy policy.

5. Data Storage & Security

  • Local data (history, API keys, settings) is stored in Chrome’s sandboxed chrome.storage.local.
  • API keys are encrypted using AES-256-GCM via the Web Crypto API before storage.
  • Backend communication uses HTTPS exclusively.
  • Server-side secrets are stored as environment variables and are never exposed to the client.

6. Permissions Explained

  • activeTab: Access the current tab’s content only when you click the extension icon.
  • storage: Store settings, history, and encrypted keys locally.
  • scripting: Inject the content extraction script into the active tab on demand.
  • alarms: Periodically refresh your authentication session in the background.

7. Data Retention & Deletion

Local explanation history can be cleared at any time via Settings → Clear All Data. Uninstalling the extension removes all locally stored data. Server-side usage logs are retained for billing purposes and can be deleted upon request.

8. Children’s Privacy

The Extension is not directed at children under 13. We do not knowingly collect personal information from children.

9. Changes to This Policy

We may update this policy from time to time. Material changes will be communicated through the extension’s update notes.

10. Contact

If you have questions about this privacy policy, please contact us at support@ailearningcopilot.com.
